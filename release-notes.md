# v1.2.1 更新内容

## 新增
- **特效工坊** — 管理后台新增视觉特效面板，支持 6 种特效：果冻模式、炫光模式（带强度滑块）、屏幕震动、3D 倾斜、CRT 扫描线、樱花飘落
- **版本更新检查** — 页面加载时自动对比 GitHub 最新版本，有更新时顶部显示提示条，支持展开查看更新日志，点击跳转 GitHub 下载
- **更新日志规范** — 版本发布时附带结构化中文更新说明

## 优化
- **数据库性能** — ApiLog 表新增 createdAt、hit、source 索引，解决大量日志数据下查询缓慢的问题
- **统计接口容错** — 缓存统计和日志统计改为独立请求，单个接口失败不影响其他数据显示
- **配置页加载** — 首次加载失败后自动重试，避免进入配置页显示空白
- **更新检查代理** — 版本检查请求自动走用户配置的 HTTP 代理，解决无法直连 GitHub 的问题
- **数字显示** — 统计数字超过 1000 显示为 k/m 单位（如 12.5k、1.3m），数据库大小超过 1000MB 自动切换为 GB
- **更新日志滚动条** — 隐藏滚动条但保留滚动功能

## 移除
- 移除物理模拟（Matter.js）
- 移除代码雨、黑洞模式
- Docker 镜像拉取指令从 release notes 中移至 README
